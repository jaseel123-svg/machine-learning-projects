# Stock Price Prediction using Machine Learning

This project predicts future stock prices using historical stock market data.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Project Workflow
1. Load historical stock dataset (Apple stock prices)
2. Preprocess and scale data
3. Train a Linear Regression model
4. Evaluate model performance using R² score and Mean Squared Error
5. Predict the next 5 days of stock prices
6. Visualize results using Matplotlib

## Files
- stock_price_prediction.py : Main machine learning model
- Apple.csv : Dataset used for training
